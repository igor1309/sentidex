🤩 Claude научился создавать и редактировать реальные файлы: excel, word, powerpoint - мой детальный обзор

Пока фича доступна только на Max, Team и Enterprise тарифах (Pro получат скоро), я сразу потестил и хочу рассказать, как это вообще работает и что можно делать.

На видео мой подробный тест этой фичи. Начал с простого - попросил создать NDA документ. Claude справился превосходно, выдал готовый Word-файл со всеми нужными секциями. Под капотом использовал JS, хорошо держит стили при апдейтах. Потом решил задать задачу посложнее: создание таблицы с юнит экнономикой для SaaS-стартапа. Claude на этот раз заюзал питон с pandas и другими либами, работал над этим довольно долго (минуты 3-5), но результат впечатлил - Excel с несколькими табами, графиками, все формулы связаны между собой и работают. Файл спокойно открывается в Google Sheets, можно сразу играться с числами, все формулы работают.

Дальше опросил на базе этого спредшита создать презентацию и заодно использовать web search для добавления рыночных данных. Claude нашел актуальные бенчмарки по SaaS-метрикам, создал презентацию по всем канонам, с ключевыми метриками, графиками из Excel. И все это можно редактировать прямо в Google Slides. Wow!

**Как это технически устроено**

Anthropic дали Claude доступ к приватному компьютерному окружению - они называют его Claude Computer. По сути это полноценный [sandbox](https://t.me/nobilix/51) с Ubuntu, где Claude может писать и запускать код, устанавливать пакеты из npm или pip, использовать специализированные библиотеки для работы с документами. В процессе видно как он тестит результат, если находит ошибки, сам фиксит.

**Неочевидные use cases**

YouTube-скрейпер через API. Claude может написать скрипт, который через Google API собирает данные о видео канала, анализирует просмотры, создает Excel с трендами и рекомендациями по контент-стратегии.

Анализ транскриптов встреч. Кидаешь запись встречи, Claude парсит кто что говорил, считает engagement каждого участника, выделяет action items, создает структурированный отчет с визуализацией "кто сколько говорил".

Генерация обучающих материалов. Из одного текстового описания курса Claude создает: презентацию для тренера, workbook для участников, чек-листы, флешкарты для запоминания, тесты с автоматической проверкой.

А еще насколько удобнее станет готовить коммерческие предложения и презы, которые потом можно редактировать в популярных форматах!

Совет: комбинируйте с [Projects](https://t.me/nobilix/154) в Claude. Загрузите туда свои шаблоны, брендбук, примеры хороших документов, он будет использовать их как референс при создании новых файлов.

Canvas и артефакты - это круто для экспериментов и быстрых набросков. Но когда нужно сделать реальную работу и передать результат коллегам или клиентам - дайте мне нормальный файл, который откроется везде. И похоже, индустрия это наконец начинает понимать. Совместимость решает.
