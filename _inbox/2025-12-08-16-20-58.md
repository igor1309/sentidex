✴️ **Skills из Claude Code — теперь в любой AI-IDE**

Систему **Skills** вынесли в универсальный **open-source** инструмент **OpenSkills**. Теперь **не нужен Claude Code** — **те же Skills** можно подключать **напрямую** в **Cursor**, **TRAE** и других IDE, включая **установку официального репозитория Skills от Anthropic в один клик**. **OpenSkills (GitHub**b](https://github.com/numman-ali/openskills)**)**

Идея простая: **Skills** становятся обычными, переносимыми «пакетами» с инструкциями и утилитами, которые можно **синкать и обновлять** без привязки к конкретному продукту. В итоге получаете **единый стек навыков** для разных IDE и одинаковое поведение агентов в вашем окружении.

**Как начать**
 🟡 Установите CLI: `npm i -g openskills`
 🟡 Поставьте официальные навыки: `openskills install anthropics/skills`
 🟡 Синхронизация: `openskills sync`
