- Save output to clearly named files (e.g. summary_2025-01.csv, cleaned_data.xlsx) in the same folder or a new 'output' subfolder
- If visualization is useful (charts, simple plots), offer it optionally with matplotlib and save as PNG
- End with a short "How this script helps" paragraph the user can read

- Assume the user knows how to install packages
- Use advanced or uncommon libraries
- Write one-liners that are hard to understand
- Leave file paths as generic variables without explanation
