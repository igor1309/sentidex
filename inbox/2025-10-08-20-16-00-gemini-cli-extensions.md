Google выпустила расширения для Gemini CLI.
[https://blog.google/technology/developers/gemini-cli-extensions/](https://blog.google/technology/developers/gemini-cli-extensions/)

Это готовые интеграции с внешними инструментами, расширенный MCP. Каждое содержит "плейбук" - инструкции, которые учат AI использовать новые инструменты. 

Пока есть несколько партнёрских 
Dynatrace (анализ производительности), Elastic (поиск данных), Figma (генерация кода из дизайна), Harness (AI для CI/CD), Postman (управление API), Shopify (работа с документацией), Snyk (безопасность), Stripe (работа с API).

И от Google:
Cloud: Cloud Run, GKE, gcloud, Observability
Разработка: Code Review, Security, Maps, Flutter, Chrome DevTools, Firebase
AI/данные: Nano Banana (изображения), Looker, BigQuery, базы данных

Ну и любой сервис может опубликовать свое расширение. Сортируют в каталоге по количеству звезд.
[https://geminicli.com/extensions/browse/](https://geminicli.com/extensions/browse/)

Это все, конечно, крайне полезно. Пусть llm-ка разбирается с разными кривыми апишками сама.
