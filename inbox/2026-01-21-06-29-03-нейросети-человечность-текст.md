📛 **Как реально сделать текст нейросети человечным**

Я постил много промптов для нейронок чтобы текст был **более "живым",** все они убирают какие-либо **паттерны**, которые чат-боты используют при генерации ответов

Но знали ли вы, что на Википедии есть **официальный список признаков сгенерированного текста?** А он есть, вот [тут](https://ru.wikipedia.org/wiki/%D0%92%D0%B8%D0%BA%D0%B8%D0%BF%D0%B5%D0%B4%D0%B8%D1%8F:%D0%9F%D1%80%D0%B8%D0%B7%D0%BD%D0%B0%D0%BA%D0%B8_%D1%81%D0%B3%D0%B5%D0%BD%D0%B5%D1%80%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%BD%D0%BE%D1%81%D1%82%D0%B8_%D1%82%D0%B5%D0%BA%D1%81%D1%82%D0%B0#%D0%A7%D1%80%D0%B5%D0%B7%D0%BC%D0%B5%D1%80%D0%BD%D1%8B%D0%B9_%D0%B0%D0%BA%D1%86%D0%B5%D0%BD%D1%82_%D0%BD%D0%B0_%D1%81%D0%B8%D0%BC%D0%B2%D0%BE%D0%BB%D0%B8%D0%B7%D0%BC%D0%B5_%D0%B8_%D0%B2%D0%B0%D0%B6%D0%BD%D0%BE%D1%81%D1%82%D0%B8) можно посмотреть, прикольно на самом деле, ко всем пунктам есть примеры

На основе этой статьи сделали **Skill** для Claude, который **исправляет текст** и убирает эти признаки - **Github**b](https://github.com/blader/humanizer)

Там всё максимально просто, но вот **процесс установки** для тех, кому лень заходить на Гитхаб:
```shell
mkdir -p ~/.claude/skills
git clone https://github.com/blader/humanizer.git ~/.claude/skills/humanizer
mkdir -p ~/.claude/skills/humanizer
cp SKILL.md ~/.claude/skills/humanizer/
```

После установки просто попросите Клода очеловечить текст, либо используйте `/humanize` вместе с вашим текстом
