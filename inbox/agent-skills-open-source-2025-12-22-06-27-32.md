**Прошло всего 3 дня с момента, как появился ****репозиторий Agent Skills**s](https://github.com/agentskills/agentskills)**, ознаменовавший переход Skills от Anthropic в OpenSource. **

Сейчас многие компании добавляют поддержку опубликованного Open Source формата. Это хорошо, так как после выхода Skills от Claude Code многие начали в спешке делать у себя "что-то своё". Теперь появилась унификация.

Ранее схожий процесс уже проходил протокол MCP, получивший виток развития после публикации в Open Source.

**Поддержку Agent Skills уже добавили:**

1. [Cursor](https://cursor.com/docs/context/skills)

2. [Codex CLI](https://developers.openai.com/codex/skills/)

3. [VS Code](https://code.visualstudio.com/docs/copilot/customization/agent-skills) и [Github Copilot](https://docs.github.com/copilot/concepts/agents/about-agent-skills) (и там, и там используют GitHub Copilot, но анонсы разные)

4. [Amp](https://ampcode.com/news/agent-skills) (изначально делали копию, которая должна быть совместима с Agent Skills)

5. [Goose](https://block.github.io/goose/docs/guides/context-engineering/using-skills/) (что-то я подзабыл про него уже)

Учитывая повсеместную поддержку [Agent Skills](https://agentskills.io/), стоит ожидать в ближайшем будущем ещё больше полезных скиллов, которые упростят работу с AI-агентами. В первую очередь, вижу преимущество для не-технических пользователей.
